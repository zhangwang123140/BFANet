# Data availability

The original image dataset is not distributed in this repository. Part of the dataset contains web screenshots or other third-party image materials for which redistribution rights are not held by the authors. Some field-collected images may also contain farm- or region-specific scene information.

The repository contains the source code, model definition, configuration file, split-file examples, and dataset-format documentation. Access to restricted image data should follow the data availability statement in the manuscript and institutional data-management requirements.
